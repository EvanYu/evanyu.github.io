Windows开机启动vmware的虚拟机

2019-03-30
Software
约 102 字 预计阅读 1 分钟

文章目录

  • 建立一个D:\Green\StartDebian.bat的脚本：
"D:\Program Files (x86)\VMware\VMware Player\vmrun.exe" start "D:\VM\Debian\Debian11.vmx" nogui
  • 开始->运行shell:startup打开启动文件夹并新建StartDebian.vbs以开机启动并隐藏启动窗口
set ws=WScript.CreateObject("WScript.Shell")
ws.Run "D:\Green\StartDebian.bat",0
